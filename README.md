# Releases of Custom ROMs
Here you will get Builds for Sweetin/RMX1851 devices

## How in Flash?
- Download Rom in your external storage
- Go to Recovery Mode
- Flash the Rom
- Format Data
- Reboot
- Booted!!

## Don't wanna Clean Flash?
- Download Rom in your Download Folder
- Go to Recovery Mode
- Flash Rom
- Reboot device
- Booted
